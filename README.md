# Pageloader.js

This plugins is written in ES6.

## HOW TO USE

### 1. Import jQuery
Add this code in your HTML

```html
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<script src="js/pageloader.js"></script>
```


### 2. Call script

```script
$(document).ready(function() {
    $('body').pageloader({
        loaderStyle: 'circle',
});
```
